# Roket Booking Project

## Project Description
The Roket Booking project is a web-based platform designed to facilitate efficient and seamless booking experiences for users. The system leverages modern web technologies to provide an intuitive interface and robust backend support, ensuring a reliable and user-friendly experience. It aims to streamline booking processes for various services.

## Implemented Functionalities
- User Authentication: Secure login and registration system.
- Booking Management: Create, view, edit, and delete bookings.
- Search Functionality: Easily search for available bookings or services.
- Responsive Design: Optimized for mobile and desktop use.

### Completed Features:
 - **User Authentication**: 
    - User registration with email validation. 
    - Secure login and logout functionality. 
    - Password hashing and recovery system.
 - **Court Booking Management**: 
    - Real-time court availability display. 
    - Ability to select courts, time slots, and dates.
    - Dynamic update of slot statuses (Available/Occupied). 
- **Payment Integration**: 
    - Simulated payment processing for court bookings. 
    - Confirmation popup after successful payment.
 - **Search Functionality**: 
    - Search courts by date, availability, or user preferences. 
- **Responsive Design**:
    - Optimized for desktop and mobile devices.

## Usage Details
- Accessing the System:
  - Navigate to the homepage to browse available services.
  - Use the search bar to find a specific court.
  - Log in or register to create an account  and manage bookings.
